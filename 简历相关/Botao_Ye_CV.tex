\cvsection{Research Experience}
\newpage

\entrytitle{CuraView: Medical Hallucination Detection and GraphRAG-Based Evidence Verification}
\entrydate{2025.04 - 2026.01}
\begin{itemize}
  \item Designed a multi-agent framework for sentence-level hallucination detection in medical discharge summaries, focusing on factual consistency risks in high-stakes clinical text generation.
  \item Built a patient-level GraphRAG pipeline over diagnoses, medications, laboratory results, timestamps, numerical records, negation relations, and contradictory evidence.
  \item Developed a seven-category hallucination taxonomy and an E1-E4 evidence grading scheme to support interpretable review and downstream training.
  \item Combined Qwen3-14B fine-tuning, structured JSON output, and Pydantic validation to improve robustness in batch inference.
\end{itemize}

\vspace{4pt}
\entrytitle{Craft-in-the-Loop: Narrative Schema-Based Multi-Agent Generation for Long-Form Text}
\entrydate{2025.06 - 2026.02}
\begin{itemize}
  \item Proposed a Planner/Writer architecture that converts narrative schema into chapter plans, stasis/process statements, and hierarchical event trees.
  \item Built a reference corpus of 100 bestselling novels and generated 100 long-form texts exceeding 100,000 tokens for comparative evaluation.
  \item Designed a six-dimensional evaluation framework covering originality, diversity, reader engagement, plot consistency, characters and settings, and spatiotemporal clarity.
  \item Showed that structured mid-level narrative control reduces long-form drift and homogenization compared with vanilla generation baselines.
\end{itemize}

\vspace{4pt}
\entrytitle{Decathlon VOC Analyzer: Evidence-Driven Multimodal VOC Analysis System}
\entrydate{2025.07 - 2026.05}
\begin{itemize}
  \item Developed a multimodal RAG pipeline for aligning reviews, product text, product images, and local visual regions with claim-level evidence attribution.
  \item Implemented aspect-level review modeling, question planning, text/image retrieval, reranking, structured report generation, and artifact-based replay.
  \item Built reproducibility support through evidence packs, retrieval logs, report claims, sidecars, and manifests.
  \item Established validation coverage across normalization, modeling, retrieval, report generation, and workflow entry points, reaching 166 automated tests.
